Engineering for Seamless UX
And Beautiful UI

안녕하세요,
매끄러운 사용자 경험 설계와 아름다운 인터페이스를 구현을 좋아하는
웹 프론트엔드 개발자 김난현입니다.

The Beginning

웹에이전시에서 UX/UI 기획자로 근무하며 여러 분야의 웹/앱 서비스의 정보구조(IA) / 인터페이스 / 와이어프레임을 설계한 경험이 있습니다.
UX/UI에 대한 고민의 결과물을 직접 구현하여 사용자에게 전달하고 싶다는 생각에 프론트엔드 개발을 시작하게 되었습니다.

Wanna Do

기술과 서비스가 사용자와 만나는 순간을 즐거운 경험으로 만들고 싶습니다.
CSS Animation, WebGL 같은 웹 그래픽스로 몰입감있는 인터랙션을 구현하는 것에 관심이 많으며, 변화에 유연하게 대응할 수 있는 아키텍처와 디자인 패턴, 테스트 코드에도 관심이 있습니다.

Wanna Be

빠르게 변하는 기술의 홍수 속에서 본질을 놓치지 않는 개발자로 성장하고자 합니다.
명확한 문제 정의와 해결을 위해, 필요한 기술을 빠르게 습득하고 적용하기 위해 방송대 컴퓨터과학과에 편입하여 CS 기본기를 쌓고 있습니다.

1

FOCUS ON THE USERS

기획 업무를 하며 갖추게 된 사용자 중심의 사고는 프론트엔드 개발을 할 때에도 사용자 관점에서 고민하는 태도와 디테일한 부분을 놓치지 않는 꼼꼼함의 기반이 되었습니다.
문제 해결을 위한 더 나은 방법을 위해 의견을 제시하며, 사용자 인터뷰를 요청하여 문제를 파악하고 개선한 경험이 있습니다.

2

MAKE IT DONE

주어진 일을 해결하기 위해 새로운 기술을 적극적으로 학습하고 적용합니다.
GraphQL, Apollo, Serverless Framework, DynamoDB 등 새로운 기술들을 습득하여 프로젝트를 성공적으로 완수한 경험이 있습니다.

3

WRITE RELIABLE CODE

적정 수준의 테스트를 통해 신뢰할 수 있고 쉽게 변경할 수 있는 코드를 작성하기 위해 노력합니다.
Jest, Testing Library, MSW, Cypress를 사용하여 단위 테스트, 통합 테스트, E2E 테스트를 작성한 경험이 있습니다.
