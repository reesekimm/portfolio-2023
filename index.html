Wanna know about me?
아름다운 인터페이스와 직관적인 사용자 경험을 구현하는 웹 프론트엔드 개발자 김난현입니다.

웹에이전시에서 기획자로 근무하며 웹/앱 애플리케이션의 정보구조(IA), 인터페이스, 와이어프레임을 설계한 경험이 있습니다. 기획 업무를 통해 '사용자 중심의 사고'를 갖게 되었고 이는 프론트엔드 개발을 할 때에도 사용자 관점에서 고민하는 태도와 디테일한 부분을 놓치지 않는 꼼꼼함을 가지는데 긍정적인 요소로 작용하고 있습니다.

빠르게 변하는 기술의 홍수 속에서 본질을 추구하는 개발자로 성장하고자 합니다. 명확한 문제 정의와 해결을 위해, 상황에 따라 적합한 기술을 찾고 빠르게 습득하기 위해 CS 기본기를 다지고 있습니다. 방송대 컴퓨터과학과에 편입하여 전공 교과목 공부와 정보처리기사 자격증 공부를 병행하고 있습니다.

그밖에 변화에 유연하게 대응할 수 있는 아키텍처와 디자인 패턴, 그리고 테스트 코드 작성에 관심이 있습니다.

1

FOCUS ON THE USERS

개발의 결과물로 사용자가 기뻐할때 가장 큰 성취감을 느끼고 동기부여를 받습니다.
개발에 임할때 항상 사용자 입장에서 고민하며 더 좋은 해결방안을 위해 적극적으로 의사소통 합니다.

2

MAKE IT DONE

주어진 일을 해결하는데 필요한 기술을 찾고, 학습하고, 적용하는것에 두려움이 없습니다.
이전 회사에서 CMS 신규 개발을 담당하여 GraphQL, Apollo, Serverless Framework, DynamoDB 등을 새롭게 학습 및 적용하고 프로젝트를 성공적으로 완수한 경험이 있습니다.

3

WRITE RELIABLE CODE

적정 수준의 테스트 코드를 통해 신뢰할 수 있는 애플리케이션을 개발하기 위해 노력합니다.
